# 📊 E-commerce Sales Dashboard (Power BI)

## 🚀 Project Overview
This project focuses on analyzing e-commerce sales data and building an interactive dashboard using Power BI.

The dashboard provides insights into sales performance, profit trends, and regional distribution.

---

## 📁 Dataset
- Source: Provided during live training session
- Format: Excel (.xlsx)
- Contains: Orders, Sales, Profit, Shipping Cost, Category, Market, City

---

## 📊 Dashboard Features
- KPI Cards (Sales, Profit, Quantity, Shipping Cost)
- Sales by Country
- Sales by City
- Sales by Category
- Sales by Market
- Sales by Shipping Mode
- Interactive Filters (Category & Market)

---

## 🛠 Tools & Technologies
- Power BI
- Microsoft Excel
